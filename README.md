# RedBull-Project

## Overview
This project is a showcase website for Red Bull drinks, highlighting their flavors, nutritional information, and marketing campaigns. The site is designed to provide an engaging user experience while promoting the iconic brand.

---

## Features
- **Drink Showcase**: A gallery of Red Bull drink varieties with images and descriptions.
- **Nutrition Information**: Detailed nutritional facts for each drink.
- **Energy Insights**: A section explaining the benefits of Red Bull as an energy drink.
- **Promotions**: Highlighted campaigns, offers, and partnerships.
- **Responsive Design**: Optimized for desktop and mobile devices.

---

## Technologies Used
- **HTML5**: Structuring content.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactive features and dynamic content.
